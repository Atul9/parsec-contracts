# Plasma MVP

[![Build Status](https://travis-ci.org/maticnetwork/plasma.svg?branch=master)](https://travis-ci.org/maticnetwork/plasma)

Install dependencies with

```
$ npm install
```

Run test cases:

```
# start test rpc server (which starts server on localhost:8545)
$ npm run testrpc

# run test cases
$ npm run test
```
